**아두이노 라이브러리 확장 for mBlcok**
본 확장은 아두이노 IDE(sketch)에서 사용하던 각종 라이브러리와 함수를
아두이노 기본 mBlock 확장과 호환하여 사용할 수 있도록 만들었습니다.

아래는 모듈과 함수 입니다.

 -LCD I2C 인터페이스------------ LiquidCrystal_I2C.h
 -로터리 엔코더----------------- Encoder.h
 -DS1302 RTC-------------------- DS1302.h
 -DHT 시리즈 온습도 센서-------- DHT.h
 -스텝모터---------------------- stepperMulti.h
 -LM35 온도센서(라이브러리 불필요)
 -TM1637 FND 모듈--------------- TM1637Display.h
 -HC-06 블루투스 슬레이브 모듈-- SoftwareSrial.h
 -GY-521 MCU6050---------------- Wire.h
 -NeoPixel----------------------Adafruit_NeoPixel.h

 -map()
 -문자열 저장
 -저장한 문자열 불러오기
 -문자열 -> 숫자 변환
 -숫자 -> 문자열 변환
